##  how to use

1. install the theme (you can sideload or grab it from Open VSX)
2. hit `Ctrl+Shift+P` or `Cmd+Shift+P`
3. run `Preferences: Color Theme`
4. select **Purpley**
5. stare into the cozy purple void
